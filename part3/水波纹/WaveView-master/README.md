# WaveView
Android自定义控件实现动态百分比水波纹效果，支持修改波纹长度、波峰高度、颜色、是否需要边框、边框的颜色、大小、可以控制波纹的移动速度，支持矩形和圆形两种效果。所有属性可以通过布局文件设置，也可以在代码中设置。

博客说明地址：<http://blog.csdn.net/itrenj/article/details/53874219>
